# v0.2.3 Testing Checklist

## Tag System
- [ ] **Create Note with Tags**: Open Create Note dialog, add 2-3 tags, create note
- [ ] **Tag Autocomplete**: Type partial tag name, verify suggestions appear
- [ ] **Create New Tag**: Type a brand new tag name, press Enter, verify it's created
- [ ] **Notebook Display**: Verify tags appear below note titles in sidebar (max 2 shown + count)
- [ ] **Edit Note Tags**: Open Edit dialog for existing note, add/remove tags, save
- [ ] **Tag Persistence**: Create note with tags, refresh page, verify tags still there
- [ ] **Keyboard Navigation**: Use arrow keys in tag autocomplete, press Enter to select
- [ ] **Tag Filtering Notebook**: Select tags in Notebook filter, verify notes filter correctly
- [ ] **Tag Filtering Graph**: Select tags in Graph filter, verify nodes highlight correctly

## Bonus Rewards System
- [ ] **Trailblazer Bonus**: Create first atom for a new text → verify +50 XP, +10 Engagement SP

- [ ] **Daily Streak**: Create atoms on consecutive days → verify streak count increases
- [ ] **Streak Multiplier**: Reach 3-day streak → verify 1.1x multiplier notification
- [ ] **Streak Multiplier 7-day**: Reach 7-day streak → verify 1.2x multiplier
- [ ] **Streak Multiplier 14-day**: Reach 14-day streak → verify 1.5x multiplier
- [ ] **Streak Reset**: Skip a day → verify streak resets to 1
- [ ] **Multiple Bonuses**: Create atom that triggers both Trailblazer and Scholar → verify both bonuses awarded
- [ ] **Bonus Notifications**: Verify toast notifications appear for each bonus type
- [ ] **Bonus Database**: Check `bonus_rewards` table for correct records
- [ ] **Streak Database**: Check `streaks` table for correct streak tracking
