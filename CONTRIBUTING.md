# Contributing to Codex Machina

Thank you for your interest in contributing to Codex Machina!

## Getting Started

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally.
3.  **Create a new branch** for your feature or bugfix.
    ```bash
    git checkout -b feature/my-new-feature
    ```
4.  **Install dependencies**:
    ```bash
    npm install
    ```
5.  **Set up your environment**:
    -   Copy `.env.local.example` to `.env.local`.
    -   Fill in your Supabase and Google Gemini keys.

## Development

-   Run the dev server: `npm run dev`
-   Ensure no linting errors: `npm run lint`

## Pull Requests

1.  Push your branch to your fork.
2.  Open a Pull Request against the `main` branch of this repository.
3.  Describe your changes and why they are needed.
